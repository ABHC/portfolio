# README

## English

### Portfolio Website

This project contains the source code for my personal portfolio. It is built with SvelteKit and includes various sections describing my work, projects, and skills.

### Features

* Modern SvelteKit architecture
* Responsive design
* Multilingual support
* Automatic deployment workflow

### Installation

```bash
# Clone the repository
git clone git@github.com:ABHC/portfolio.git
cd portfolio

# Install dependencies
npm install

# Run in development
npm run dev
```

### Build

```bash
npm run build
```
